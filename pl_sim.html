Full Season Simulator

38 gameweeks, from scratch · Poisson goal model · synthetic double round-robin
No real fixture list was supplied, so the schedule is synthetic: a standard double round-robin for the 20 clubs (everyone plays everyone home and away once, 38 gameweeks total, 10 matches per week). The whole season is simulated from 0 points — current standings are used only to estimate each club's underlying attack and defense strength (goals scored/conceded per game), which drive the Poisson goal model for every match. This is not the real fixture list, and the simulated table will diverge from the actual current table since it's an entirely fresh random season built on the same underlying team strengths.
